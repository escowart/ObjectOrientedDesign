
Class: Object Oriented Design - Fall 2013 - Northeastern University

The folders in this repository contain all the src files and extra files for each of the homeworks. homeworks 1 through 3 are all fundamental programs while homeworks 4 - 8 are all different stages of a Binary Search Tree which started as a Binary Search Tree of a specific type and ended up being a Red-Black Tree that self balanced with a parameterized type, a iterator over the tree, and a visitor pattern to operate over the tree.
